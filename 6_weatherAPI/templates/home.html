Usage

To retrieve the weather data, you should send a GET request to one of the following endpoints:

  • To get all the data available:
    http://127.0.0.1:5000/api/station_id
  • To get the data for specified date (YYYY-MM-DD):
    http://127.0.0.1:5000/api/station_id/date
  • To get the data in between specified dates (YYYY-MM-DD):
    http://127.0.0.1:5000/api/station_id/from_date/to_date

For example:
http://127.0.0.1:5000/api/26/2010-01-23
Will return the output:
{ "date": "2010-01-23", "station": "26", "stationName": "Polis", "temperature": 14.6 }

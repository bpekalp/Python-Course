Usage

To retrieve the words and descriptions, you should send a GET request to the following endpoint:
http://127.0.0.1:5000/api/word/id

For example:
http://127.0.0.1:5000/api/word/1
Will return the output:
{ "description": "A departure from what is normal, usual, or expected, typically an unwelcome one.", "id": 1, "word": "Aberration" }
